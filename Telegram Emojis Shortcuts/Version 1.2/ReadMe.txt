This app allows you to send emojis using the function keys (F1, F2, ...) in Telegram.

-How to use:
  Download the "Telegram Emojis Shortcut V1.2.exe" file to use the app. No instalations is needed, you only need to run the application.
  The app does not have an user interface, but you will see an icon in the taskbar.
  
  Shortcuts:
    Press F1 to F11 to insert emojis.
    Press Shift +  F1 to F11 to send the emoji as normal emoji instead of an animated sticker.
    Press Ctrl + Shift + F1 to F11 to insert 4 emojies at once.
    Press F12 to send a desired emoji you've picked, as a normal emoji instead of an animated sticker.

  The hand emoji's color are the colors that you've chosen in the emoji menu. 



-If you want the app to automatically run on windows start-up, do the steps below:
  -Open "Run" by pressing win + r
  -Type in "shell:startup" & press enter
  -Paste the app in the folder that is opened



-Source Code:
  The "Telegram Emojis Shortcut V1.2.ahk" file is the source code. This app is written used the AutoHotKey script language; so you'll need to install AutoHotKeys to be able to edit the source code. The app itself how ever, does not need any extra instalations.
